# Silky Marketing Studio

A modern marketing studio website built with Next.js, React, and TypeScript.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To create a production build:

```bash
npm run build
```

To start the production server:

```bash
npm start
```

## Technologies

This project is built with:

- Next.js 15
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion
